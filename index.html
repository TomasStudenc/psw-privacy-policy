Privacy Policy

PSW Password Saver respects user privacy.

Data Collection

The application does not collect, transmit, or share any personal or sensitive user data. The application works fully offline and does not require an internet connection.

Data Storage and Security

All user data is stored locally on the user's device only. Sensitive information is encrypted using AES encryption. Stored data is saved as encrypted PNG files within the application's private storage or cache.

Access Protection

Access to the application and stored data is protected using a master password or biometric authentication supported by the device.

Third-Party Access

No third parties have access to user data. The application does not use analytics, advertising SDKs, or online services.

User Control

Users can permanently delete all stored data by clearing the application data or uninstalling the application.
