# psw-privacy-policy
Privacy Policy

PSW Password Saver respects user privacy.

Data Collection:
The application does not collect, store, transmit, or share any personal or sensitive user data with the developer or third parties.
The application works fully offline and does not require an internet connection.

Data Storage and Security:
All user data is stored locally on the user's device only.
Sensitive information is encrypted using AES encryption.
Stored data is saved as encrypted image (PNG) files within the application's private storage or cache.

Access Protection:
Access to the application and stored data is protected by a master password and/or biometric authentication supported by the user's device.

Third-Party Access:
No third parties have access to user data.
The application does not use analytics, advertising SDKs, or online services.

User Control:
All stored data can be permanently removed by clearing application data or uninstalling the application.
